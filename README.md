# ⌨️ Apprendre le clavier

Petits outils pédagogiques, gratuits et hors-ligne, pour apprendre l'informatique en douceur.
Conçus pour de grand·e·s débutant·e·s, en français, clavier **AZERTY**.

## Contenu

### 🎮 Apprendre le clavier — `Apprendre-le-clavier.html`
Un logiciel d'entraînement à la frappe, inspiré de Ratatype.

- **16 leçons progressives** (2 à 4 touches à la fois) + **3 défis vitesse** chronométrés
- Clavier AZERTY affiché à l'écran : la touche à frapper **s'allume en jaune**, passe au **vert** si c'est la bonne, au **rouge** sinon
- Indication du **doigt** à utiliser pour chaque touche
- Précision, vitesse (mots/min), chronomètre et **étoiles**
- **Tableau de bord** de progression + sauvegarde automatique (dans le navigateur) + bouton de remise à zéro

### 🖨️ Fiche raccourcis Windows — `Fiche-raccourcis-Windows.html`
Une fiche aide-mémoire **prête à imprimer** (A4, une page) avec les raccourcis Windows du quotidien,
classés par couleur et pensés pour une lecture facile.

## Utilisation

Aucune installation. **Double-cliquez** sur le fichier `.html` voulu : il s'ouvre dans le navigateur.
Pour imprimer la fiche : `Ctrl + P`.
